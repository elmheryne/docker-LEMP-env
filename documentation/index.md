Documentation for docker-LEMP
==========================================================

## Usage :

- for Nginx alone                      : run-LE
- for Nginx + PHP                      : run-LEP
- for Nginx + PHP avec phalcon         : run-LEP-phalcon
- for Nginx + PHP + mysql              : run-LEMP
- for Nginx + PHP avec phalcon + mysql : run-LEMP-phalcon
