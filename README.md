Docker dev platform - Version 1.0
=============================================================

	- based on webofmars images (std debian jessy)
	- nginx
    - php5-fpm
	- mysql


*Usage :*

run "docker-compose -f docker-lnp.yml" for Nginx + PHP-FPM
run "docker-compose -f docker-lnpm.yml" for Nginx + PHP-FPM + Mysql
